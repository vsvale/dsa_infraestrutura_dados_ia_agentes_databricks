# Política de Privacidade - LGPD Compliance

## 📋 Informações Gerais

Esta plataforma **Serasa Experian Analytics** foi desenvolvida em conformidade com a **Lei Geral de Proteção de Dados (LGPD)** - Lei 13.709/2018.

### 🎯 Finalidade
- Análise de dados financeiros de mercado
- Fornecer insights e visualizações de ações
- Educação e pesquisa sobre investimentos
- Nenhuma informação pessoal é coletada ou armazenada

## 🔒 Dados e Privacidade

### Dados Coletados
- **NENHUM** dado pessoal é coletado
- Apenas dados públicos de mercado são utilizados
- Símbolos de ações são informações públicas
- Dados históricos são provenientes de fontes públicas (Yahoo Finance)

### Dados Não Coletados
- ❌ CPF/CNPJ
- ❌ Nome completo
- ❌ Endereço
- ❌ Email
- ❌ Telefone
- ❌ Qualquer informação identificável

## 🛡️ Medidas de Segurança

### Criptografia e Armazenamento
- Dados em cache por 5 minutos apenas
- Nenhuma informação pessoal persistida
- Conexões via HTTPS obrigatórias
- Logs sem informações sensíveis

### Monitoramento e Auditoria
- Logs de acesso anonimizados
- Monitoramento de performance
- Auditoria de segurança contínua
- Backup automático de configurações

## 📊 Direitos dos Titulares

### LGPD - Seus Direitos
1. **Confirmação de Existência** ✅
   - Confirmamos que NÃO processamos dados pessoais

2. **Acesso aos Dados** ✅
   - Todos os dados são públicos e acessíveis

3. **Correção de Dados** ✅
   - Dados podem ser atualizados em tempo real

4. **Anonimização** ✅
   - Nenhum dado pessoal para anonimizar

5. **Portabilidade** ✅
   - Dados são exportáveis a qualquer momento

6. **Eliminação** ✅
   - Cache limpo automaticamente a cada 5 minutos

7. **Informação sobre Compartilhamento** ✅
   - Nenhum dado é compartilhado com terceiros

## 🚀 Práticas de Desenvolvimento

### Privacy by Design
- Arquitetura focada em não coleta de dados pessoais
- Revisão de código para verificação LGPD
- Testes automatizados de privacidade
- Documentação transparente

### Segurança Implementada
- Validação de entrada de dados
- Proteção contra injeção de código
- Rate limiting para requisições
- Headers de segurança HTTP
